triangularmap.tmap.dict_tmap

triangularmap.tmap.dict_tmap(n, default_factory, *args, **kwargs)[source]

Create a TMap of size n using a defaultdict as underlying storage. Elements will only be initialised (using the default_factory function) when accessed. Note that pretty() printing the map will typically access (and thus initialise) all elements.

Parameters:

  • n – size of the map

  • default_factory – function to initialise values in the dict/map (passed on to defaultdict as first argument)

  • args – additional arguments passed on the defaultdict

  • kwargs – additional arguments passed on the defaultdict
